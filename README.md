# MyBear
Java高仿FastDFS
#### 如何在IDE中开发
在src\test\resources\storage.conf这个文件中的base_path和store_path0写上本地路径
例如:


store_path0 = d:/fastdfs

base_path = d:/fastdfs


之后使用src\test\java\io\mybear中的测试类启动

##### header

| 位置   | 内容       |
| ---- | -------- |
| 0~7  | 包总长      |
| 8    | 命令       |
| 9    | 状态（0为正确） |
